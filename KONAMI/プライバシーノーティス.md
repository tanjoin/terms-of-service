<h1>
プライバシーノーティス<strong>［My KONAMIサービス］</strong>
</h1>

<p>
株式会社コナミデジタルエンタテインメント（Konami Digital Entertainment Co., Ltd.）（以下、「<strong>当社</strong>」といいます。）は、お客様がKONAMI IDによる認証を用いるサービス（以下、「<strong>My KONAMI</strong>サービス」といいます。）を利用する場合におけるお客様の個人データ（識別可能な個人としてのお客様に関するあらゆる情報をいいます。）について、適用される法令及びガイドライン等に基づき、以下のとおり、取り扱います。
</p>
<p>
別紙に定める国・地域の法令が適用される場合には、それぞれ該当する別紙が、本プライバシーノーティスと併せて適用されます。本プライバシーノーティスと別紙との間に矛盾がある場合には、別紙の内容が優先するものとします。
</p>
<p>
当社は、本プライバシーノーティスの一部を適宜変更、追加又は削除することができます。本プライバシーノーティスの最新版は、当社のウェブサイト https://legal.konami.com/kde/id/view/ja/ から、本プライバシーノーティスの過去の版は https://legal.konami.com/kde/id/archive/view/ja/ から、それぞれ入手可能です。

<strong>8. 安全管理措置</strong>
</h2>
<p>

</p>
<ul class="a">
<li>
<strong>個人データの取扱いに係る規律の整備</strong><br> 当社は、個人情報の保護に関する基本方針としてプライバシーポリシーを策定し、また、お客様の個人データの取得、利用、保存、提供、削除・廃棄等に関して、従業員に適用される個人データの取扱いに関する規程を策定しています。当社のプライバシーポリシーは https://legal.konami.com/en/ で確認いただけます。
</li>
<li>
<strong>組織的安全管理措置</strong>
<ul class="k">
<li>
安全管理措置を組織的に講じるために、コナミグループ株式会社情報セキュリティ委員会を中心とした体制を構築し、安全管理措置の評価及び見直し・改善を年次で行っています。

<li>
個人情報取扱台帳（Record of Processing Activities）を整備しています。
</li>
<li>
個人データや情報システムの外部持出に関する規則を整備し、実施しています。
</li>
<li>
個人データを取扱う業務を委託する際は、安全管理措置に関する条項を含めるとともに、安全管理措置は年次で評価及び見直し・改善を行っています。
</li>
</ul>
</li>
<li>
<strong>人的安全管理措置</strong>
<ul class="k">
<li>
全ての従業者との間で、採用・契約の際に、非開示契約を締結しています。非開示条項は、契約終了後も一定期間有効です。
</li>
<li>
全ての従業者に対し、個人情報の取扱いに必要かつ適切なトレーニングを年次で実施しています。
</li>
</ul>
</li>
<li>
<strong>物理的安全管理措置</strong>
<ul class="k">
<li>
個人データを取扱う業務は、入退館が管理され、かつ物理的に保護された区画で行っています。個人データを取扱う情報システムは、保護された区画に設置されています。
</li>
<li>
個人データを取扱う区画への入退館は、監視カメラにより常時記録され、区画の出入り口は適切な施錠が行われています。
</li>
<li>
当該区画においては、外部記録媒体の利用は原則的に禁止されています。
</li>
</ul>
</li>
<li>
<strong>技術的安全管理措置</strong>
<ul class="k">
<li>
個人データへのアクセスには、正当なアクセスであることを確認するために、アクセス権限を有する従業者本人であることの識別と認証が必須となっています。
</li>
<li>
個人データへのアクセスは識別に基づいたアクセス制御が行われており、従業者にアクセス権限を付与する際には、最小化の原則に従い、業務に比例した付与者数と権限付与を適切に管理し実施しています。
</li>
<li>
個人データ含めたデータの送受信は、適切な等価安全性（equivalent security）で暗号化されています。また、個人データを取扱う情報システムの使用状況やアクセス状況は、一部の操作内容を含めて監視を行っています。
</li>
<li>
個人データを取扱うシステムは、会社が管理するデバイスのみとなっており、適切なセキュリティアップデートを実施しています。また、合理的かつ有効なマルウェア対策や不正ソフトウェア対策も併せて実施しています。

</ul>
</li>
</ul>

<h2>
<strong>4. PDPAに基づくお客様の権利</strong>
</h2>
<p>
お客様は、お客様の個人データに関して、以下に説明する法律上の権利を有しています。
</p>
<ul class="h">
<li>
<strong>個人データにアクセスする権利</strong>：お客様はいつでも、ご自身の個人データのコピーを提供するよう当社に請求する権利を有します。この権利にはいくつかの制限があるため、当社はお客様が請求する情報の全てをお客様に提供できない場合があります。
</li>
<li>
<strong>個人データの訂正を求める権利</strong>：お客様はいつでも、ご自身の個人データが正確ではなかった場合はこれを訂正し、完全ではなかった場合はこれを補完するよう当社に請求する権利を有します。
</li>
<li>
<strong>個人データの消去を求める権利</strong>：状況によっては、お客様はご自身の個人データを削除するよう当社に請求することができます。
</li>
<li>
<strong>個人データの利用の制限を求める権利</strong>：状況によっては、お客様はご自身の個人データの当社による使用を制限するよう当社に請求することができます。
</li>
<li>
<strong>データポータビリティについての権利</strong>：状況によっては、お客様はご自身の個人データについて、かかるデータの保存及びアクセスのために一般的に使用されている形式で別の組織又はご自身に移転するよう当社に請求することができます。この権利は、お客様が当社にご自身の個人データを提供し、さらに、当社がお客様の個人データを使用する法的根拠がお客様の同意又は契約のいずれかである場合にのみ適用できます。
</li>
<li>
<strong>個人データの使用について異議を申し立てる権利</strong>：状況によっては、お客様は、当社によるお客様の個人データの使用に同意しないことを主張することができます。この権利は、当社がお客様の個人データを使用する法的根拠が正当な利益である場合に適用されます。お客様からご自身の個人データを当社が使用することに対する異議を受け取った場合、当社は一般にその使用を中止します。ただし、当社にその使用についての強い理由があり、当社の継続的な使用がお客様に容認できないリスクを生じさせない場合に限って、当社はその使用を継続することができます。もっとも、お客様の異議申立てがダイレクトマーケティングに対するものである場合には、当社は常に使用を中止します。
</li>
<li>
<strong>お客様による同意の撤回</strong>：お客様の個人データを使用する当社の法的根拠に同意された場合、お客様はいつでも考え直してご自身の同意を撤回する権利を有します。
</li>
</ul>
<p>
以上の権利行使については、以下のとおり処理されます。
</p>
<ul class="h">
<li>
<strong>対応までの所要時間</strong>：当社は、個人データの請求に対して30日以内に対応するよう努めています。場合によっては、（例えば、お客様の請求が複雑な場合、又は、多数の請求をされた場合等）30日以上かかることがあります。また、上記のとおり、適用される法令に基づいてお客様が有する法律上の権利には例外もありうるため、当該権利が適用されない場合があることにご留意ください。
</li>
<li>
<strong>お客様に関する追加情報</strong>：当社は、お客様の身元確認のために、お客様に追加情報を求める必要があることがあります。これは、個人データを受け取る権利のない人に個人データが提供されないようにするためのセキュリティ対策です。また、お客様の請求を明確にするためにお客様にご連絡することがあります。
</li>
<li>
<strong>手数料</strong>：お客様は、ご自身の個人データにアクセスするため、又はその他の権利のいずれかを行使するために、通常はお支払いの必要はありません。ただし、お客様が特に不当な請求をされていると当社が判断した場合等においては、当社は相応な手数料を請求することがあります。一例として、お客様が不当に多数の請求をしている場合、又は、不当に大量の情報を請求している場合等がこれに当たります。また、このような場合には、お客様の請求に応じられない場合があります。
</li>
</ul>
